> 이 문서는 개발 진행에 따라 계속 수정되는 **작업용 기획안(Single Source of Truth)** 입니다.

## 2. 게임 의도 및 핵심 가치

- 거울의 방향을 조절하여 레이저를 목표 지점까지 정확히 반사시키는 과정에서 **반사 방향 예측, 공간지각, 경로 추론, 문제 해결 능력**을 기른다.
- 단순 조작이 아니라 **"어느 거울을 어떤 순서로 돌려야 하는가"** 를 판단하는 사고 과정이 핵심이다.

## 3. 게임 구성 요소

| 구성 요소 | 설명 |
|-----------|------|
| 레이저 발사 장치 | 레이저를 발사하는 시작 지점 (점등 효과 적용) |
| 목표 지점 | 레이저가 도달해야 할 최종 지점 (점멸 효과 적용) |
| 거울 | 회전시켜 레이저 경로를 바꾸는 핵심 오브젝트 (조작 가능, 은은한 강조 표시) |
| 고정 돌 블록 | 통과 불가 장애물 (움직일 수 없음) |
| 금지 구역 | 레이저가 닿으면 즉시 실패 처리되는 구역 |
| 이동 장애물 | 일정한 패턴으로 움직이는 장애물 (고난도 적용) |
| 특수 장치 | 스테이지가 높아질수록 추가되는 추가 요소 |

## 4. 게임 진행 방식 (5단계)

### STEP 1. 맵 확인 및 예측 단계
- 게임은 총 3개 라운드로 진행되며, 각 라운드는 같은 난이도의 두 스테이지로 구성된다.
- 두 팀의 순서를 정해 번갈아 가며 라운드를 진행한다.
- 화면에 레이저 발사 장치, 거울, 장애물, 목표 지점이 나타난다.
- 라운드 시작 전 5~10초 동안 전체 맵이 정지된 상태로 보인다.
- 학생들은 현재 배치된 거울과 장애물 위치를 확인하고, 팀별로 레이저 경로를 예측한다.
- 이 단계에서 실제 레이저는 아직 발사되지 않으며, 전략을 세우는 준비 단계이다.

### STEP 2. 팀 협의 및 거울 조작 단계
- 팀 대표를 정하거나 화면을 터치할 순서를 정한다.
- 제한 시간 내에 팀원들과 예측하고 협의한 내용을 바탕으로 거울을 회전시킨다.
- 기본 회전 단위는 **90도**이며, HARD 모드에서는 **45도** 단위 회전도 가능하다.
- 거울 터치 시 "딸깍"과 같은 회전 사운드 효과가 적용된다.
- 선택된 거울은 외곽선이 강조되어 표시된다.
- 남은 시간이 카운트다운으로 표시된다.

### STEP 3. 조작 완료 및 최종 확인
- 팀이 거울 조작을 완료하면 완료 버튼을 누른다.
- 제한 시간이 초과되면 현재 상태로 자동 진행된다.

### STEP 4. 레이저 발사 및 결과 확인
- 완료 버튼을 누르거나 제한 시간이 초과되면 자동으로 레이저가 발사된다.
- 발사 장치에서 나온 빛이 거울에 반사되며, 실제 이동 경로가 한 줄씩 순차적으로 시각화되어 보인다.
- 레이저가 장애물에 막히거나 잘못된 방향으로 나가면 **"실패"** 로 처리된다.
- 레이저가 목표 지점에 정확히 닿으면 **"성공"** 으로 처리된다.

#### 점수 체계

| 결과 | 조건 | 점수 |
|------|------|------|
| 완벽 성공 | 모든 거울 사용 + 목표 도달 | 10점 |
| 부분 성공 | 일부 거울만 사용 + 목표 도달 | 5점 |
| 실패 | 목표 미도달 / 장애물 충돌 / 금지 구역 진입 | 0점 |
| **콤보 보너스** | 레이저가 거울을 연속으로 통과할 때마다 누적 (2콤보부터 가산) | +1점 × (콤보 단계 − 1) |

> 예: 거울 4개를 모두 거치며 목표 도달 → 완벽 10점 + 콤보(2·3·4 → 1+2+3) 6점 = **16점**

### STEP 5. 난이도 상승 및 최종 승부
- 라운드가 진행될수록 거울 수, 장애물 종류, 특수 장치 등이 추가되어 난이도가 높아진다.
- 점수는 스테이지를 거치며 누적되고, 실패해도 다음 라운드로 이어진다.
- 3개 라운드 완료 후, 누적 점수가 높은 팀이 승리한다.

## 5. 난이도 확장 요소

| 확장 항목 | 내용 |
|-----------|------|
| 거울 수 증가 | 스테이지가 올라갈수록 조작해야 할 거울 개수가 늘어남 |
| 회전 각도 세분화 | 기본 90도 → HARD 모드에서는 45도 단위 회전 가능 |
| 중간 목표 추가 | 목표 지점이 2개로 분할 (1번 지점을 거쳐 2번 지점 도달 등) |
| 시간 단축 | 스테이지가 올라갈수록 제한 시간이 짧아짐 |
| 이동 장애물 추가 | 일정 패턴으로 움직이는 장애물 등장 |
| 특수 장치 도입 | 금지 구역 등 추가 요소가 상위 스테이지에서 적용됨 |

## 6. 시각 및 음향 효과 요구사항

### 시각 효과
- 난이도 선택 화면 (EASY / NORMAL / HARD) 제공
- 레이저 발사 장치 점등 효과
- 목표 지점 점멸(블링킹) 효과
- "STAGE START" 과 같은 느낌의 짧은 시작 효과
- 선택 가능한 거울 위에 회전 아이콘 또는 노란 방향 화살표 표시
- 조작 가능한 오브젝트만 은은하게 강조 표시
- 선택된 거울 외곽선 강조
- 레이저 경로가 한 줄씩 순차적으로 그어지며 나타남
- **거울 통과 콤보 연출** — 레이저가 거울에 반사될 때마다 해당 거울 위치에서 콤보 텍스트(`COMBO x2`, `x3`, …) 팝업과 링/파티클 폭발이 발생하고, 콤보 단계가 올라갈수록 색상이 변화 (노랑 → 주황 → 붉은 금색)
- 성공 시 목표 지점 점등 + 점수 획득 효과
- 실패 시 붉은 경고 표시 (네비게이션 과속 화면과 같은 느낌)
- 남은 시간 카운트다운 표시

### 음향 효과
- 거울 터치 시 "딸깍" 회전 사운드
- 거울 통과(콤보) 시 반사음 — 콤보가 누적될수록 피치가 점점 올라감
- 성공 시 성공 사운드
- 실패 시 짧은 실패 음향 효과

### 선택 사항 (있으면 좋음)
- 한 번에 여러 거울을 동시에 터치 가능한 기능

## 10. 개발 메모 (구현 결정 사항)

> 이 섹션은 개발하면서 채워 나가는 기술적 결정 기록입니다.

### 기술 구성
- **플랫폼**: 전자칠판 → 웹 기반 (HTML5 Canvas + JavaScript, 빌드 도구 없음). `index.html`을 브라우저로 열면 바로 실행.
- **입력**: `pointerdown` 이벤트로 터치/마우스 동시 지원.
- **그리드**: 거울/장애물/발사장치/목표는 격자(grid) 셀 단위 배치. 화면 크기에 맞춰 셀 크기 자동 조정.

### 게임 메커니즘
- **거울 4종**: `/` `\`(대각선), `|` `-`(가로/세로). EASY·NORMAL은 90도 회전(`/`↔`\`), HARD는 45도 회전(4종 순환).
- **레이저 시뮬레이션**: 격자 광선 추적. 셀 단위 진행, 거울에서 반사·통과 판정.
- **레벨 생성**: 절차적 생성 — 경로를 먼저 만든 뒤 거울 방향을 흐트러뜨림 → **정답이 항상 존재**함을 보장 (720개 생성 검증 완료).
- **진행**: `intro → predict → manipulate → fire → result` 5단계 phase 상태 머신.
- **점수**: 완벽 성공 10 / 부분 성공 5 / 실패 0. 팀별 누적, 6스테이지(3R×2) 후 최종 승부.

### 파일 구조
```
index.html          진입 HTML (화면 마크업)
css/style.css        스타일
js/config.js         난이도·라운드 파라미터
js/audio.js          WebAudio 합성 효과음
js/laser.js          레이저 광선 추적
js/levels.js         절차적 스테이지 생성기
js/render.js         Canvas 렌더링
js/game.js           게임 상태 머신·진행 로직
js/main.js           진입점·렌더 루프
```

### 구현 완료 (v1)
- 타이틀/난이도 선택, 게임 진행 5단계, 결과·최종 화면
- 거울 회전, 레이저 추적·경로 순차 연출, 점수·팀 교대
- 발사장치 점등 / 목표 점멸 / 거울 강조 / 회전 아이콘 / 카운트다운
- 효과음(딸깍·발사·반사·성공·실패), 설정(소리 ON/OFF)

### 구현 완료 (v2 — 콤보 연출)
- 레이저가 거울에 반사되는 시점마다 콤보 카운트 +1
- 거울 위치에서 `COMBO x2/x3/…` 텍스트 팝업이 위로 떠오르며 페이드
- 콤보 단계별 색상 변화: 2 노랑 → 3 주황 → 4 붉은 금색 → 5+ 금색
- 반사 지점 링·파티클 폭발 연출
- 콤보 단계가 올라갈수록 반사음 피치 상승 훅(현재 사운드 자체는 비활성)
- 콤보 보너스 점수: 2콤보부터 (단계−1)점씩 누적, 결과 카드에 별도 표시

### 미구현 / 향후 확장 (기획안 5·6절)
- 이동 장애물 (일정 패턴으로 움직이는 장애물)
- 중간 목표 (목표 지점 2개 분할: 1번 경유 → 2번 도달)
- 여러 거울 동시 터치
- 점수판(🏆) 버튼 기능 — 현재 효과음만 (상세 화면 미구현)
